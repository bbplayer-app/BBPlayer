# 排行榜

既然我们记录了你的播放数据，那么生成一个排行榜自然是理所应当的事情。

## 播放统计

BBPlayer 会根据你的本地播放记录，统计出你听歌最多的曲目和总时长。

### 访问方式

在「音乐库」点击右上角的 **排行榜** 图标（奖杯形状）。

### 功能说明

- **听歌排行**：展示你所有歌曲的播放次数排行。点击歌曲可以直接播放。
- **总时长统计**：页面顶部会显示你使用 BBPlayer **完整播放**歌曲的总时长。

> [!NOTE]
> 这里统计的“完整播放”是指歌曲从头播放到尾。切歌或未播放完则不会计入时长统计。
