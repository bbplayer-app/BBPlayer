# 设置与个性化

每个人都有自己的听歌习惯，BBPlayer 提供了丰富的设置选项来满足你的需求。

## 播放设置

在 **设置 -> 播放设置** 中，你可以调整：

- **恢复播放进度**：开启后，应用启动时会尝试恢复上次关闭时的播放进度。
- **启动时自动播放**：配合上一条使用，打开 App 就接着听（小心社死）。
- **响度均衡 (实验性)**：尝试利用 b 站的 API 数据将所有歌曲的音量统一到一个标准水平，避免切歌时音量忽大忽小。（可能会导致音质略微下降）
- **歌词源**：选择自动匹配歌词的来源（网易云/QQ音乐/自动）。

## 桌面歌词 (Android)

在 **设置 -> 播放设置** 中开启。
开启后，歌词会以悬浮窗的形式显示在屏幕最上层。你可以自由拖动位置，或者在设置中将其 **锁定** 以防止误触。

> [!IMPORTANT]
> 启用桌面歌词需要授予「悬浮窗」权限。首次开启时，应用会引导你前往系统设置页面进行授权。
